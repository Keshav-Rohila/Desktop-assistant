Functions defined ----------------------------------------------------------------------

   1. wishMe() function returns wishes 'Good Morning' , 'Good Afternoon' , 'Good Evening'
               according to the current time.

   2. speak() function speaks the string argument passed in the function 
               and it also prints the string on the output screen.

   3. takeCommand() function takes the command from the user in form of text or speech.

   4. check_internet_connection() function basically checkes the internet connection and
                                   returns True or False.

   5. filter_string() function

   6. check_on_wikipedia() function
                     



Features added to this virtual assistant ------------------------------------------------

   1. It takes voice as well as text input from the user.
   2. It wishes us greeting at the time of execution according to the current time.
   3. It opens google in defalut browser if we command "open google".
   4. It checks the internet net connectin on command "check internet connection".
